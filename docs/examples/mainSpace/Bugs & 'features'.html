Known issues, features and bugs to be addressed in librat


Issue
Cause
Divide by zero due to a 1/cos(zenith) in the geometry.location & geometry.lookat

Effect
rat: signal interrupt 11 for process 47658
(try switching it off and on again ;-)

Fix: trap better in code

Issue
Cause
Number of groups permitted is hardwired to 50000. This is problem if you have many large object files

Effect
rat: signal interrupt 11 for process 47658
(try switching it off and on again ;-)

Fix:
Can be overridden by setting environment variable MAX_GROUPS i.e. setenv MAX_GROUPS 200000 say. Other environment variables used in librat:

MAX_SUNS (default = 180)
PRAT_MAX_MATERIALS (default = 1024)

Issue
Non-Lambertian material properties not accounted for in spectral invariant mode

Fix: pass through in code
